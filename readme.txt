1. py folder contains all python programs
2. source folder contains all images used in application
3. txt folder contains files 
4. uploads is the folder when you select files to upload in the index.html page
5. index.html is the home page to select files to upload
6. ocr.php is the second page which index.html page linked to when click on the submit button
7. Usage: user click on select buttons from each category and upload files by click on submit button;
		users are directed to ocr.php page, click on image to Text conversion button, it will process file and click on submit;
		users are deirected to textProcess.html page, click on big Submit and it pops up the processed file names, then click on submit to go to next step;
		users are directed to outcome.html page, click on Data Extraction, the extracted data will pop up
